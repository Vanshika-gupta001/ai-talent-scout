# ai-talent-scout
# 🛡️ RE-SCREEN AI: Neural Resume Screening Terminal

**RE-SCREEN AI** ek advanced NLP-powered tool hai jo Job Descriptions (JD) aur Resumes ke beech ke gap ko analyze karta hai. Ye tool recruiters ko hazaron resumes mein se top candidates select karne mein madad karta hai using Vector Logic aur Neural Extraction.

🚀 **Live Demo:** https://resumes-screening-sysytem.streamlit.app/

---

## ⚡ Key Features
* **Neural Extraction:** Spacy NLP model ka use karke skills extract karna.
* **Vector Matrix Mapping:** TF-IDF aur Cosine Similarity ka use karke accuracy score nikalna.
* **Cyberpunk UI:** Ek modern, neon-themed dashboard for better user experience.
* **Skill Gap Analysis:** Instant visualization ki candidate mein kaunsi skills missing hain.
* **Radar Charts:** Candidate ki competency ka graphical representation.

---

## 🛠️ Tech Stack
* **Frontend:** Streamlit (Python-based Web Framework)
* **NLP:** Spacy (`en_core_web_sm`)
* **Analytics:** Scikit-learn (TfidfVectorizer), Pandas, NumPy
* **Visualization:** Plotly, Altair
* **PDF Parsing:** PyMuPDF (Fitz)
